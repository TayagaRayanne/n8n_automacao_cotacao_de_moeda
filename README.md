# n8n_automacao_cotacao_de_moeda
O fluxo pesquisa a cotação da moeda, realiza a conversão e gera alertas por e-mail de acordo com valores mínimos de referência. Quando os valores são mais altos que o limite, o sistema salva automaticamente os dados em uma planilha do Google Sheets, registrando a cotação, a data e a hora, criando assim um histórico detalhado de picos de valor.
